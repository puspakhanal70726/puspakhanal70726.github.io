How I monitor my Project?

He said that he would help me, and he explained it to me for 30 minutes, but I was still not fully confident; I just knew a little bit, but it was alright. Something was better than nothing.

On the second day, I first made the HTML and CSS directories, and I started working with the HTML. It took almost all of the hours in classes to completed the files. I also used some CSS files, with the basics. I used the GitHub page for the peer review. I got a comment on the HTML, and I needed to fix some spelling errors and attributes. It took 10-15 minutes to fix that comment because I had sent a giant HTML file for the peer review. After I finished fixing the comment, I started working on the CSS, and when the class was over, I packed my class materials and went home.
